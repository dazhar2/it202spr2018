Answers to Questions

  1. It is the CSS box model. In this model, for every HTML element, there is a box that wraps around it, consisting of margins, borders, padding, and, finally, actual content.
  2. A block-level element always starts on a new line and takes up the full width available, creating a "block".
  3. The paragraph tag: <p>
  4. Inline elements are those which only occupy the space bounded by the tags, rather than interrupting the flow of the content on a new line.
  5. The span tag: <span>
  6. An external style sheet (in link elements) or an internal style sheet (between style tags).
  7. Selectors find html elements to attribute the style to those elements that are tagged with those specific selectors.
  8. body {background-color: #0000FF;}
